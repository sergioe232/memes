Before we start...

When building a website, it is important to understand the basic level functionality of how a website works and what it needs to function appropriately. 

A website runs on something called a server. A server is the physical machine computer that hosts your website's content and distributes it to people all over the world. One of the issues in owning a server or renting a server a server from a third-party is that servers are expensive :(

A website also needs code in order for the server to properly display and distribute content to the users. Reinventing the wheel by developing code that will do this is unfortunately very time consuming, and again expensive :(

Luckily, we do not have to do any of this. There are countless third-party website hosting companies like Wordpress, Wix, Weebly, and many other platforms that provide excellent professional website building tools that cut out the need to code everything.

On top of that, many of these platforms will host your website on their servers entirely FOR FREE.
